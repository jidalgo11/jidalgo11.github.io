Terminology

Animations

CSS allows animation of HTML elements without using JavaScript or Flash.

Keyframes

@keyframes controls the intermediate steps in a CSS animation sequence by defining styles for keyframes.

Transforms

The transform property applies a 2D or 3D transformation to an element. This property allows you to rotate, scale, move, etc., elements.

Transitions

The transition property allows us to change property values smoothly, over a given duration.

Easing

Easing is the way a transition effects moves.

Rotation

With the transform property we can rotate elements.

Scale

With the transform property we can scale elements.

Skew

With the transform property we can scale elements.

Delay

We can control when a transition effect will start with transition-delay.

Transition Timing

The transition-timing-function specifies the speed curve of the transition effect.
